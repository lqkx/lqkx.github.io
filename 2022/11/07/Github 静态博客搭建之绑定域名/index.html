Github 静态博客搭建之绑定域名

Created2022-11-07|Updated2022-11-08|hexo博客搭建
|Word count:336|Reading time:1min|Post View:
购买域名

本人是在腾讯云买的，按照官网流程购买并实名认证即可，完成后的界面如下。

配置域名解析

鼠标悬停至DNSPod,出现一个对话框，点击“管理解析”跳转至管理界面。

管理界面如下，点击快速添加解析

在这里，显然我们使用的是GitHub的服务器，那么只要ping一下Github生成的项目静态网址即可获取该服务器主机的ip地址，如下图所示。

将IP地址填入，点击确定，可得两行记录

现在在浏览器输入lqkx.club显然是不行的，因为还未设置域名映射。观察到如下说明

注意A记录是将域名指向一个IPv4地址，上一步已经完成，而CNAME就是域名映射。所以只需要将第二行的域名记录改成CNAME,值设置为项目静态网址即可。如图

在博客项目目录的source文件夹下添加CNAME文件

在source文件夹下新建一个CNAME文本，打开并输入个人域名（lqkx.club），保存

删除.txt后缀，保存。

在Github绑定域名

输入项目网址

点击“Settings”

点击“Pages”

下滑，在自定义域名这块输入自己的域名。可以选择强制https

绑定成功
